# RGB Dot Matrix Desk Display

A 64x32 RGB LED matrix display project for Raspberry Pi, with a web-based simulator for testing code before deployment.

## Features

- 64x32 LED matrix simulator in the browser
- Real-time weather display for NYC
- Easy testing before deploying to Raspberry Pi
- GitHub-connected for easy updates

## Quick Start

1. Clone this repository
2. Open `index.html` in your web browser
3. The simulator will display the weather for NYC

## Future Plans

- Deploy to Raspberry Pi with RGB LED matrix
- Add more display modes (clock, notifications, etc.)
- Custom animations and transitions

## Hardware

- 64x32 RGB LED Matrix Panel
- Raspberry Pi
- Power Supply (5V 4A recommended)
